# IDC (Invasive Ductal Carcinoma) Detection Model

- most common type of breast cancer, originates in the milk ducts and spreads into surrounding breast tissue

## Approach of this project will be following:

- used dataset will be split into 70:30:10 ratio, respectively image set for training, image set for fine-tuning and for final evaluation
- first I will work on unbalanced data, analyse the results, after that with balanced data as the whole dataset is quite unbalanced
